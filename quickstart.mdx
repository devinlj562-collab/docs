## Set up locally

1. Clone your docs repository from your [dashboard](https://dashboard.mintlify.com)
2. Install the CLI: `npm i -g mint`
3. Run `mint dev` in your docs directory
4. Open `http://localhost:3000`

## Make your first edit

1. Open `docs.json`
2. Update `"name"` and `"colors"` fields
3. Save and preview changes at `http://localhost:3000`

## Deploy

1. Install the [GitHub app](https://dashboard.mintlify.com/settings/organization/github-app)
2. Commit and push your changes
3. Your docs deploy automatically
